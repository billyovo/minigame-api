record

Get records by server

Get records by server

path Parameters
server
required
string
Enum: "survival" "skyblock" "all"

Search record in this server

query Parameters
before
string
Example: before=63fa06c748fe037985ba5fc4

Get records before this id

after
string
Example: after=63fa06c748fe037985ba5fc4

Get records after this id

dateBefore
string
Example: dateBefore=2023-02-28

Get records before this date with format yyyy-MM-dd

limit
integer
Example: limit=20

Maximum number of record to return in one query

Responses

https://minigame-api.letsdream.today/record/{server}

Content type
application/json
